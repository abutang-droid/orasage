## 2. 标志系统（方案 A「玄璧」）

### 2.1 构成与寓意

标志由两部分构成：

1. **图形标（玄璧 Aperture）**：一枚带缺口的圆环 + 中心一点。
   - **缺口** = Oracle：天机进入的通道，置于东北 45° 方位（紫气东来）。
   - **中心一点** = Sage：观照后的笃定。
   - 圆环同时是 OraSage 的字母 **O**，图形可独立使用。
2. **字标（Wordmark）**：`OraSage`，思源宋体 SemiBold（600），字距 `0.08em`（顶栏场景延续现有 0.08–0.1em 实现）。

### 2.2 图形标几何规格（canonical）

画布 `viewBox="0 0 64 64"`，圆心 `(32,32)`：

| 元素 | 规格 |
| :--- | :--- |
| 圆环 | 半径 22，描边 4.5，`stroke-linecap="round"` |
| 缺口 | 圆心角 38°，中心方位 45°（屏幕东北）；弧线自 64° 逆时针绕行至 26° |
| 中心点 | 半径 4.5 实心圆 |
| 颜色 | 单色：墨 `#171717`（正）/ 纸 `#FAFAF8`（反白） |

**标准 SVG 源（勿手改路径数值）：**

```svg
<svg viewBox="0 0 64 64" fill="none" xmlns="http://www.w3.org/2000/svg">
  <path d="M 41.645 12.226 A 22 22 0 1 0 51.774 22.355"
        stroke="#171717" stroke-width="4.5" stroke-linecap="round"/>
  <circle cx="32" cy="32" r="4.5" fill="#171717"/>
</svg>
```

**小尺寸光学修正**（渲染尺寸 = 图形标整体呈现宽度）：

| 渲染尺寸 | 描边 / 中心点半径 |
| :--- | :--- |
| ≥ 48px | 4.5（标准） |
| 32–47px | 5.5 |
| 24–31px | 6.5 |
| 16–23px（favicon 下限） | 8 |

### 2.3 使用形态：默认分开，组合仅限正式场合

**基本原则（2026-07-08 确认）：图形标与字标默认分开使用，各司其职；禁止把「图形 + 字标」组合锁定强行铺到所有位置。** 界面日常出现的品牌，是字标或图形其中之一，不是两者并排。

| 形态 | 构成 | 场景 |
| :--- | :--- | :--- |
| **字标独立（界面默认）** | 仅 `OraSage` 衬线文字 | PC / 移动顶栏、页脚法律行、正文行内提及 —— 现有纯文字顶栏**保持不变** |
| **图形独立（符号场景）** | 仅玄璧 | favicon、App 图标、底栏图标、水印、社交头像、印章、加载屏 |
| 横排组合（正式锁定） | 图形标 + 间距（图形直径 × 0.35）+ 字标；图形高度 ≈ 字标 cap height × 1.6，垂直居中 | **仅限**：邮件页脚、报告封面 / 封底、包装、名片、商店物料等「正式署名」场合 |
| 竖排组合（正式锁定） | 图形标居中，字标于下方，间距 = 图形直径 × 0.3 | 包装侧面、竖版海报、启动屏 |
| 副标语锁定 | 组合下方 `ORACLE + SAGE`，思源宋体 600、全大写、字距 0.32em、深灰或中灰 | 品牌页、封底、周边 |

判断口诀：**页面里已经有字标，就不要再加图形；符号位（图标、水印、印章）只放图形，不要再塞字**。同一视口内图形与字标同时出现时，两者须承担不同职能（如顶栏字标 + 底栏图形图标），不得作为装饰重复。

### 2.4 留白与最小尺寸

- **净空区**：组合四周留白 ≥ 图形标直径 × 0.5；任何文字、图形、页面边缘不得进入。
- **最小尺寸**：图形独立 ≥ 16px；横排组合宽度 ≥ 88px；印刷图形直径 ≥ 6mm。

### 2.5 颜色版本

| 版本 | 图形 / 字标 | 底色 | 用途 |
| :--- | :--- | :--- | :--- |
| 墨版（默认） | `#171717` | 纸 `#FAFAF8` / 白 `#FFFFFF` | 全站、印刷正文 |
| 反白版 | `#FAFAF8` | 墨 `#171717` | 墨底锁定（邮件页脚、报告封底、包装贴纸） |
| 朱砂底版 | `#FAFAF8` | 朱砂 `#A63F33` | 节庆 / 祈福限定 App 图标、封口贴纸 |
| 朱砂印版 | `#A63F33` | 纸 / 白 | 仅作「钤印」语义的小尺寸印记（见 §3.2 配额） |

### 2.6 误用禁例

- 禁止拉伸、旋转（缺口方位固定东北 45°）、加投影 / 渐变 / 描边外发光。
- 禁止改变缺口位置或角度、增删中心点。
- 禁止用灰度以外的颜色（朱砂仅限 §2.5 所列两种版本）。
- 禁止将图形标与其它图形拼合成新标志。
- 禁止在复杂照片上直接放置墨版标志（需衬纸色或墨色底块）。

## 3. 色彩系统

### 3.1 灰度宪法（沿用 DS v1.1，不变）

| Token | Hex | 用途 |
| :--- | :--- | :--- |
| 墨 Ink | `#171717` | 品牌、一级文字、主按钮 |
| 深灰 Deep Gray | `#6B7280` | 二级文字、图标 |
| 中灰 Mid Gray | `#A1A1AA` | 占位符、弱文字 |
| 浅灰 Light Gray | `#E7E5E4` | 边框、分割线 |
| 纸 Paper | `#FAFAF8` | 全局背景 |
| 宣白 White | `#FFFFFF` | 卡片、输入框 |

### 3.2 朱砂印色（v1.0 新增）

| Token | Hex | 用途 |
| :--- | :--- | :--- |
| `--os-color-cinnabar-600` | `#A63F33` | 朱砂主色（对白底对比度 ≈ 6.2:1，可作小字） |
| `--os-color-cinnabar-700` | `#8C3529` | 朱砂元素 hover / active |
| `--os-color-cinnabar-100` | `#F6E8E4` | 朱砂徽记浅底（极少用） |

**定位：书画上的一方钤印，不是第二品牌色。**

允许出现的场景（全部满足「小面积、点睛」）：

1. 印点 / 徽记、节庆与祈福限定 App 图标；
2. 祈福・神庙的「香火」语义（香火计数、点灯状态等）；
3. 命理报告中的关键批注 / 钤印；
4. 线下物料的封口贴纸、钤印。

**禁止**：按钮、链接、导航、大面积底色、正文强调。单屏 / 单页面朱砂面积配额 **≤ 5%**。

> 保底条款：任何页面移除全部朱砂元素后，信息层级必须依然完整（朱砂只锦上添花，不承担唯一语义）。

### 3.3 功能色（沿用 status token，不变）

成功 `#16A34A` · 警告 `#D97706` · 错误 `#DC2626` · 信息 `#2563EB`。仅限表单校验与状态反馈，禁止挪作品牌装饰。

## 4. 字体系统

| 角色 | 字体栈 | 用法 |
| :--- | :--- | :--- |
| 品牌字标 | Source Han Serif SC（Noto Serif SC）600 | `OraSage` 及子品牌字标，字距 0.08em（背书行 0.14–0.22em） |
| 中文标题 H1–H4 | Source Han Serif SC Bold | 同 DS v1.1 |
| 中文正文 | PingFang SC / Noto Sans SC | 16px / 1.6 |
| 拉丁 / 数字 | Inter | 日期、价格、域名 |
| 数据 / 干支 | JetBrains Mono | 干支、时间码、坐标 |
| 拉丁展示衬线 | Source Serif 4 | 英文场景大标题（替代已退役的 Playfair Display） |

- 网页按 DS v1.1 token（`--os-font-*`）执行；印刷与社媒图使用同一家族的桌面版本。
- 全站不引入第三方展示字体；营销物料如需书法笔触，以**单色线描插画**表达而非字体。

## 6. 数字应用 · 网站

### 6.1 favicon / 图标套件（8 应用统一）

| 资产 | 规格 | 说明 |
| :--- | :--- | :--- |
| `icon.svg` | 玄璧墨版，透明底 | 现代浏览器首选 |
| `favicon.ico` | 16 + 32 双尺寸，按 §2.2 光学修正 | 兼容 |
| `apple-touch-icon.png` | 180×180，墨底反白，无圆角（系统裁切） | iOS 收藏 |
| `icon-512.png` / `icon-192.png` | 墨底反白，PWA manifest | Android |

源文件放 `shared/brand/favicon/`，同步至各 Next 应用 `app/`、bazi 的 Vite `index.html` 与 auth-service 静态页。

### 6.2 OG / Twitter 分享图

模板（1200×630）：纸底 + 左上角**字标**（衬线 24px，不加图形）+ 大号衬线标题（52–64px，最多两行）+ 左下域名（Inter 500，字距 0.08em，深灰）+ 右上角 6% 透明度大**图形**水印（出血）。字标与图形在卡片中分处两角、各司其职（§2.3），不组成锁定。

- 各 Next 应用以 `opengraph-image.tsx`（`ImageResponse`）按页面 metadata 自动生成；bazi/auth 静态页用预渲染 PNG。
- `twitter:card` 维持 `summary_large_image`。

### 6.3 壳层品牌位

- PC 顶栏 / 移动顶栏：**维持纯文字字标 `OraSage`**（§2.3 界面默认形态），不加图形标。
- 移动底栏第二键：门户上下文用玄璧**图形**替代 `LayoutGrid`（符号位只放图形）；子应用上下文维持各子标图形。
- 页脚遵守 DS v1.1 §7；墨底场景（如报告封底页）使用反白锁定（正式署名场合）。

### 6.4 邮件

- 页眉：仅字标墨版，居左（不加图形，§2.3）。
- 页脚：墨底反白锁定 + 法律行 `© {year} OraSage. All rights reserved.` + 隐私 / 条款链接（同 DS §7.2）。
- 正文遵守灰度宪法；朱砂只可作一处「钤印」级点缀（如订单完成的印点）。

### 6.5 加载 / 空态 / 404

- 全屏 Loading：纸底 + 居中玄璧，`orasage-fade-in`（motion token），禁止旋转动画（缺口方位固定）。
- 404 / 空态插画遵守 §10 线描风格；玄璧可作 8–12% 透明度背景水印。

## 7. 数字应用 · 移动 App

> 适用于未来原生 / 壳 App（iOS、Android）及 PWA。

### 7.1 App 图标

| 项 | 规范 |
| :--- | :--- |
| 主图标 | 墨底 `#171717` + 反白玄璧，图形占安全区直径 62% |
| iOS | 提供 light / dark / tinted 三态（dark 用墨底反白；tinted 提供单色模板层） |
| Android 自适应图标 | 前景层：玄璧反白；背景层：纯墨；不得依赖圆角形状 |
| 节庆 / 祈福限定 | 朱砂底 `#A63F33` + 反白玄璧，限时上架，过期回滚 |
| 子应用独立 App（如有） | 墨底 + 对应子标反白，右下角不加角标 |

### 7.2 启动屏（Splash）

纸底 `#FAFAF8`，居中竖排组合（图形 88px + 字标），淡入 450ms（`--os-ease-standard`）；禁止广告位与第三方 logo 同屏。

### 7.3 系统级图标

- Android 状态栏推送小图标：玄璧单色剪影（纯白，系统着色）。
- 快捷方式 / 3D Touch 图标：lucide 线性图标风格，描边 1.6。

### 7.4 商店物料

| 资产 | 规范 |
| :--- | :--- |
| 商店名称 | `OraSage — 命理与能量`（副标题按商店字数限制截取） |
| 截图模板 | 纸底 + 顶部衬线短句（一屏一句）+ 设备框内实机截图；每套首屏含横排组合 |
| Google Play Feature Graphic（1024×500） | 墨底反白锁定 + 副标语 `ORACLE + SAGE` |
| 预览视频 | 片头 / 片尾用 §8.4 动态锁定，正片右下 6% 透明度玄璧水印 |

### 7.5 In-App 品牌时刻

- 报告生成完成、订单支付成功等「仪式性节点」允许一处朱砂钤印动效（缩放 0.985→1 + 淡入，遵守 motion token）。
- 深色模式：App 内深色主题以 DS v1.1 dark token 为准；品牌资产用反白版。

## 8. 社交媒体

### 8.1 头像（全平台统一）

墨底 + 反白玄璧（同 App 主图标），导出 800×800 PNG。子品牌账号（如有）用对应子标同规格。**全平台头像不使用照片或插画。**

### 8.2 封面 / 主页横幅

模板：纸底或墨底 + 横排组合居左 + 副标语；不堆砌卖点文案。

| 平台 | 尺寸 | 版本 |
| :--- | :--- | :--- |
| X (Twitter) | 1500×500 | 墨底反白 |
| YouTube | 2560×1440（安全区 1546×423） | 纸底墨版 |
| Facebook Page | 820×312 | 纸底墨版 |
| 微信公众号首图 | 900×383 | 纸底墨版 + 单色线描插画 |
| 小红书 / 抖音主页背景 | 竖版 3:4 起 | 纸底竖排组合 |

### 8.3 贴文图模板

- 画幅：1:1（1080×1080）与 4:5（1080×1350）两套母版。
- 结构：纸色卡片 + 衬线大标题（≤2 行）+ 正文（Noto Sans SC，≤3 行）+ 左下**字标**（小，不加图形）+ 右下朱砂印点（可选，一图至多一枚）。
- 系列内容（如「每日一签」）：右上角深灰细体编号（JetBrains Mono）。
- 命理图表 / 干支：JetBrains Mono，禁止用花体字。
- 禁止：渐变底、贴纸风 emoji 堆叠、描边艺术字。

### 8.4 视频

- 片头（≤1.5s）：纸底，玄璧淡入 + 中心点后现（「先有环，后有点」），随后字标浮现。
- 片尾（≤2s）：墨底反白锁定 + 域名。
- 常驻水印：右下角玄璧，6–8% 透明度。
- 字幕：Noto Sans SC，白字 + 60% 墨色衬底条。

### 8.5 语气与运营边界

- 文案语气：静谧、笃定、不贩卖焦虑；避免「测测你…」式营销腔。
- 平台正文文字可按平台习惯适量使用 emoji，但**品牌图片资产内**禁止 emoji。
- 用户内容转发（命盘截图等）须打上 §8.4 水印规格。

## 9. 线下传播

### 9.1 印刷色值与纸张

| 项 | 规范 |
| :--- | :--- |
| 墨（大面积） | 四色黑 C30 M30 Y30 K100；小字 / 细线用单黑 K95 |
| 纸色 | 优先用未涂布自然白纸张呈现（≈ `#FAFAF8`），不专门印底色 |
| 朱砂 | 以 `#A63F33` 打样校准；四色近似 C0 M62 Y69 K35；批量物料建议专色（从 Pantone 174 C / 7592 C 邻近色打样确定，定稿后回写本表） |
| 纸张 | 名片 / 卡片：300g 以上未涂布（超感 / 刚古类）；册页：100–120g 书纸；包装：白卡或牛皮卡 |
| 工艺白名单 | 单色印刷、朱砂专色、压凹（玄璧盲压）、UV 局部（仅玄璧）；**禁止**烫金、镭射、渐变四色堆叠 |

### 9.2 名片 / 信纸 / 工牌

- 名片 90×54mm：正面纸色 + 墨字（姓名衬线、信息 Inter/Noto Sans）+ 右下朱砂印点；背面墨底反白居中图形标。
- 信纸 / PDF 信头：左上横排组合（高 8mm），页脚法律行同 §6.4。
- 工牌 / 挂绳：墨底反白；挂绳织带印 `ORASAGE` 全大写字标（字距 0.32em，此为唯一允许的全大写装饰场景）。

### 9.3 电商包装（水晶 / 周边）

| 层 | 规范 |
| :--- | :--- |
| 外箱（快递） | 牛皮卡单色墨印：侧面横排组合 + `orasage.com`；不印营销语 |
| 内盒 | 白卡纸色，盒面盲压玄璧（无墨压凹），侧面竖排组合 |
| 棉纸 | 纸色棉纸满版 8% 透明度玄璧阵列水印 |
| 封口贴纸 | 圆形 40mm：朱砂底 + 反白玄璧（本方案朱砂底版的主要出口） |
| 卡片 | 随盒「能量卡」：正面商品五行 / 水晶说明（线描插画），背面墨底锁定 |
| 缎带 / 束口袋 | 墨色或本白，单色印字标 |

### 9.4 命理报告印刷版

- 开本 B5 或 A4；封面：纸色 + 竖排组合 + 客户姓名（衬线）+ 日期（JetBrains Mono）。
- 内页：遵守 DS 排版层级；干支 / 星盘图单色线描；关键批注可用朱砂（每跨页 ≤1 处）。
- 封底：墨底反白锁定 + 副标语 + 域名二维码（二维码单黑，居中嵌玄璧 logo 需留白安全区）。
- 骑缝章：朱砂玄璧印（实体印章按 §2.2 几何制作，直径 20mm）。

### 9.5 海报 / 展会 / 空间

- 海报网格：留白 ≥40%，一张海报一个信息层级；玄璧可作满版 6–8% 水印。
- 展位 / 快闪：以「宣纸挂轴」为空间母题——立式长条幅（纸色 + 单列衬线大字 + 底部锁定），主背板墨底反白。
- 物料灯光：暖白 3000–3500K，禁止彩色氛围灯压过灰度体系。

### 9.6 周边（衍生品）

允许品类基调：文房（笔记本、书签、印章）、织物（束口袋、帆布袋）、香烛（祈福语义可用朱砂）。一律单色 + 至多一处朱砂；禁止满版印花。
