## 4. E2E 脚本（`scripts/e2e/`）

**一键冒烟（推荐）：**

```bash
cd scripts/e2e
npm install
npx playwright install chromium
npm run test:smoke-all
```

**执行顺序（`run-smoke-all.mjs`）：**

1. `test:platform-report` — 平台 report checkout API  
2. `test:shop-security` — BIZ-003 / BIZ-008  
3. `test:auth-internal` — BIZ-004  
4. `test:admin-gate` — TC-ADMIN-001/002  
5. `test:auth-redirect` — TC-AUTH-003  
6. `test:auth-sso` — TC-AUTH-001  
7. `test:auth-center` — TC-AUTH-002  
8. `test:tarot-offer` / `test:tarot-free` / `test:tarot-daily` / `test:tarot-temple`  
9. `test:shop-crystal` — TC-SHOP-001  
10. `test:pay-double` — UI-001  
11. `test:pay-network` — NET-002  
12. `test:shop-flow` / `test:ziwei-flow` — bazi/ziwei 浏览器闭环  
13. `verify-unify` — 各站底栏一致性  

**常用环境变量：**

| 变量 | 默认 | 说明 |
|------|------|------|
| `E2E_AUTH_URL` | `https://auth.orasage.com` | auth 基址 |
| `E2E_SHOP_URL` | `https://shop.orasage.com` | shop 基址 |
| `E2E_ADMIN_URL` | `https://admin.orasage.com` | admin 基址 |
| `E2E_PASSWORD` | `E2eTest2026!` | 自动注册测试账号密码 |
| `E2E_ADMIN_EMAIL` / `E2E_ADMIN_PASSWORD` | 未设 | admin happy path（可选） |

**Tarot 寺庙 E2E 注意：** 长按拜神需对「感受临在」的 **父节点** 做 `mouse.down/up` ≥3.5s（内层 text div 无 React handler）。

## 6. 部署备忘

- **Shop 重建**需 `source /opt/orasage/.env` 取 `JWT_SECRET`，否则 Next build 失败。  
- **命理 App** 需与 auth-service **相同 `JWT_SECRET`** 才能识别 `orasage_token`。  
- **Admin 提升**：生产手动 SQL（见 `deploy/admin/README.md`）  
  `UPDATE users SET role='admin' WHERE email='...';`  
- 本轮 **#61 纯 E2E**，无需 redeploy admin。  
- `deploy/VPS-DEPLOY.md` 中 admin 502 描述可能过期；当前 admin 生产 200。
