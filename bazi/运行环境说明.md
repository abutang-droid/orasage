# OraSage 八字计算器 - 本地运行环境检查

## 运行方式

项目使用 pnpm 管理依赖（package.json 中指定 packageManager: pnpm@10.4.1）。

```bash
pnpm install        # 安装依赖
pnpm dev            # 启动开发服务器
pnpm build          # 构建
pnpm check          # TypeScript 类型检查
pnpm test           # 运行测试
```

## 开发服务器

启动后访问: http://localhost:3000/

多语言测试:
- http://localhost:3000/ → 简体中文（默认）
- http://localhost:3000/?lang=zh-TW → 繁体中文
- http://localhost:3000/?lang=en → English
- http://localhost:3000/?lang=pt-BR → Português (BR)

## 环境变量

`.env` 文件中已配置:
- DEEPSEEK_API_KEY — AI 解读功能必需
- DATABASE_URL — MySQL（本地开发可无）
- WORDPRESS_URL — 报告推送地址
